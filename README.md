# elite Auto Protection Landing Page

This is a premium, high-conversion landing page built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

## 🛠 Manual Setup Required (Media)

As requested, all `src` attributes for images and videos have been left empty for you to fill manually. You will find placeholders in the following components:

- **Hero Section** (`src/components/Hero.jsx`): Hero video background.
- **Before & After Section** (`src/components/Comparison.jsx`): Before and After images for the slider.
- **Services Grid** (`src/components/Services.jsx`): Images for each service card.
- **Process Section** (`src/components/Process.jsx`): Visuals for each installation step.
- **Testimonials** (`src/components/Testimonials.jsx`): Thumbnail for the client story video.
- **Location Map** (`src/components/Contact.jsx`): Google Maps iframe source.

## ✨ Premium Features Included

- **Custom Luxury Cursor**: A magnetic-style cursor that interacts with the UI.
- **Grainy Texture Overlay**: A subtle semi-transparent noise layer for a tactile, premium feel.
- **Infinite Brand Marquee**: Showcasing luxury brands (Porsche, Ferrari, etc.) for social proof.
- **Scroll Progress Indicator**: A dynamic blue line at the top tracking the user's journey.
- **Pulsing Floating Contact**: Instant access to contact/WhatsApp.
- **Premium Typography**: Fully integrated "Outfit" Google Font.
- **Adaptive Glassmorphism**: Smooth, blurred backgrounds on navigation and cards.

## ⚠️ Troubleshooting

If you encounter an error like `powershell not found` when running commands, ensure that your system's `PATH` include the WindowsPowerShell directory: `C:\Windows\System32\WindowsPowerShell\v1.0`.
